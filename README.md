# FUTURE_FS_02
Client Lead Management System (Mini CRM) - Task 2
A simple Mini CRM application to manage client leads generated from contact forms.

##Features
- Add new leads
- View all leads
- Update lead status (New / Contacted / Converted)
- Delete leads
- RESTful APIs using Node.js & Express

##Tech Stack
- Backend: Node.js, Express
- Database: MongoDB
- Frontend: HTML, CSS, JavaScript
- Tools: Postman, GitHub
